# Análise de Anúncios de Venda de Carros

Este projeto é uma aplicação web desenvolvida com Python e Streamlit para analisar um conjunto de dados de anúncios de venda de carros.

## Funcionalidades

O aplicativo permite:

- Visualizar um histograma da coluna `odometer` para analisar a distribuição da quilometragem dos veículos.
- Visualizar um gráfico de dispersão entre `odometer` e `price` para observar a relação entre quilometragem e preço.
- Interagir com os gráficos usando caixas de seleção no Streamlit.

## Tecnologias utilizadas

- Python
- Pandas
- Plotly Express
- Streamlit
